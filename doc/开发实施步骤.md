## 阶段七：部署与优化（第7周）

### 7.1 Docker化
**文件**: `docker-compose.yml`
```yaml
version: '3.8'
services:
  backend:
    build: ./backend
    ports:
      - "8000:8000"
    environment:
      - CONFIG_PATH=/app/config/config.yaml
    volumes:
      - ./config:/app/config
      - ./logs:/app/logs
    depends_on:
      - postgres
      - redis

  frontend:
    build: ./frontend
    ports:
      - "3000:80"

  postgres:
    image: postgres:15
    environment:
      POSTGRES_DB: feishu_bot
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: ${DB_PASSWORD}
    volumes:
      - postgres_data:/var/lib/postgresql/data
      - ./scripts/init_db.sql:/docker-entrypoint-initdb.d/init.sql

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./config/nginx.conf:/etc/nginx/nginx.conf
    depends_on:
      - backend
      - frontend

volumes:
  postgres_data:
```

### 7.2 部署脚本
**文件**: `scripts/deploy.sh`
```bash
#!/bin/bash
# 拉取最新代码
git pull origin main

# 构建镜像
docker-compose build

# 停止旧容器
docker-compose down

# 启动新容器
docker-compose up -d

# 查看日志
docker-compose logs -f
```

### 7.3 Linux服务器部署
```bash
# 1. 安装Docker和Docker Compose
# 2. 克隆项目
git clone https://github.com/your-repo/feishu-project-bot.git
cd feishu-project-bot

# 3. 配置环境变量
cp config/config.example.yaml config/config.yaml
vim config/config.yaml  # 填入实际配置

# 4. 初始化数据库
docker-compose up -d postgres
docker-compose exec postgres psql -U postgres -d feishu_bot -f /docker-entrypoint-initdb.d/init.sql

# 5. 启动所有服务
docker-compose up -d

# 6. 配置飞书回调地址
# http://your-server-ip/api/v1/feishu/callback
```

## 技术难点与解决方案

### 1. AI解析准确性
**问题**: 日报格式多样，解析可能不准确
**方案**:
- 提供日报模板引导
- 支持手动修正
- 持续优化Prompt

### 2. 飞书API限流
**问题**: 高频调用可能触发限流
**方案**:
- 使用Redis队列缓冲
- 实现请求重试机制
- 批量处理消息

### 3. 外部人员权限
**问题**: 外部人员可能无法@机器人
**方案**:
- 申请企业外部权限
- 提供备用提交方式（Web表单）

### 4. 数据一致性
**问题**: 并发更新可能导致数据不一致
**方案**:
- 使用数据库事务
- 使用Redis分布式锁
- 记录详细变更日志
